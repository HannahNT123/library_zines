About This Zine Library

This project is a digital zine library dedicated to zines about libraries, librarians, and librarianship. It brings together zines created by and for library and information workers, with a focus on access, ethics, pedagogy, and professional practice.

The library uses a GitHub repository as its underlying infrastructure and GitHub Pages as its public interface. Zines are catalogued using structured metadata and controlled subject terms, making them browsable and searchable through tag-based filtering.

This project was developed as part of coursework in information retrieval, and also functions as an experiment in what it means to treat a Git repository as a library.
